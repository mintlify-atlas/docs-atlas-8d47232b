Before you can search for Kurals, Thirukkural.AI asks you to complete a personality quiz. Your answers shape how the platform understands you and influence the "Surprise Me" feature that returns a Kural matched to your type.

## How it works

<Steps>
  <Step title="Answer 15 questions">
    Each question has four options (A–D). You navigate forward and back freely, but must answer the current question before moving to the next.
  </Step>
  <Step title="Scores are tallied">
    Every answer adds points to one or more of five dimensions: `wisdom`, `compassion`, `strength`, `harmony`, and `ego`.
  </Step>
  <Step title="Personality type is determined">
    The dimension with the highest total score determines your type. Ego has special override logic — if it is the highest score, or is above 5 and at least as high as `strength`, the Ambitious Achiever type is assigned regardless of other scores.
  </Step>
  <Step title="Result is saved">
    Your answers are written to `localStorage` under the key `quizAnswers`. On future visits the app reads this value and skips the quiz entirely. You can retake it at any time using the "Retake Personality Quiz" link on the main page.
  </Step>
</Steps>

## Scoring dimensions

Each answer maps to one or more dimensions with a point value. The five dimensions and what they represent:

| Dimension | What it measures |
|-----------|-----------------|
| `wisdom` | Analytical reasoning, truth-seeking, ethical decision-making |
| `compassion` | Empathy, care for others, emotional attunement |
| `strength` | Courage, determination, decisiveness |
| `harmony` | Balance, peacefulness, conflict avoidance |
| `ego` | Ambition, pride, self-interest |

### Type resolution logic

```typescript
// From quizLogic.ts — analyzePersonality()
const scores = { wisdom, compassion, strength, harmony, ego };
const maxScore = Math.max(...Object.values(scores));

if (ego === maxScore || (ego > 5 && ego >= strength)) {
  return { type: 'The Ambitious Achiever', ... };
} else if (wisdom === maxScore) {
  return { type: 'The Wise Seeker', ... };
} else if (compassion === maxScore) {
  return { type: 'The Compassionate Heart', ... };
} else if (strength === maxScore) {
  return { type: 'The Strong Leader', ... };
} else {
  return { type: 'The Peaceful Soul', ... };  // harmony wins or fallback
}
```

<Tip>
  Ties between non-ego dimensions are broken by the order of the `if/else` chain: wisdom beats compassion, compassion beats strength, and strength beats harmony.
</Tip>
